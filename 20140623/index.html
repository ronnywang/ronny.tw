數學 SUM SUM(值_1, 值_2) 傳回一連串數字或儲存格的總和。 瞭解詳情
統計 AVERAGE AVERAGE(值_1, 值_2) 傳回資料集的數字平均值 (忽略文字)。 瞭解詳情
統計 RANK RANK(值, 資料, 遞增) 傳回資料集中指定值的排名。 瞭解詳情
數學 COUNTIF COUNTIF(範圍, 條件) 傳回範圍內符合條件的項目數量。 瞭解詳情

座標規則

  1. 一格: B3
    2. 第 B 行第 3 列
  2. 範圍: B3:C4
    3. B3 到 C4 之間的 2x2 格的範圍
  3. 範圍: B3:B
    4. B 整列的資料，去掉 B3 之前的值 (通常用在 B1, B2 是欄位名稱的情況)
  4. 範圍: B3:C
    5. B, C 整列的資料，去掉 B3, C3 之前的值
  5. 一格: B$3
    6. 第 B 行第 3 列，而且將這格複製到其他格時，B 會跟著平移，但是 3 會固定住
  6. 一格: $B$3
    7. 第 B 行第 3 列，而且將這格複製到其他格時， B 和 3 都會固定住
