ronny.tw/data

這幾年收集整理了很多資料，做了很多平台，但是常常到後來自己都忘了有收集過哪些資料，因此打算把做過的東西都放在這邊

平台

台灣公司資料

company.g0v.ronny.tw

資料來源為 經濟部全國商工行政服務入口網 ，更新頻率為每個月一次，包含全台灣公司名單營業項目、資本額、董監事名單，以及台灣各商號資料等

台灣公司關係圖


company-graph.g0v.ronny.tw

將台灣公司資料的公司以董監事名單連結起來，只要輸入統一編號就可以看到以該公司為中心的公司關係圖

關務署進出口資料整理


portal.g0v.ronny.tw

來自經濟部關務署的所有進出口貨物的進口出口國家、貨物項目申報的重量以及價值資料，以月為單位，並且有畫成圖表可以看出歷史的變化，資料從 2003 年開始，更新頻率為每個月

PTT熱門記錄


ptthot.ronny.tw

從 2011/11 開始收集每五分鐘抓取PTT熱門看板人氣數字，並記錄當時的板名，可以看出 PTT 各看板在各社會事件的人氣變化

NewDiff


newsdiff.g0v.ronny.tw

從 2013/8 開始收集各網路媒體的新聞內容，並且追蹤是否有被刪除或是內容修改，以方便研究新聞修改原因，持續自動更新中

NewsTrend


newstrend.g0v.ronny.tw

利用 NewsDiff 資料做的服務，可以輸入關鍵字，搜尋該關鍵字在各媒體出現的歷史記錄，以方便分析各媒體的新聞偏好以及事件的關注度

歷史四大報頭條


oldpaper.g0v.ronny.tw

收集 2011/11 開始的每日四大報頭條新聞，以方便研究時事記錄以及研究媒體偏好

台電發電量歷史記錄


taipower.g0v.ronny.tw

每十分鐘收集台電各種發電量數字，畫出歷史趨勢圖

台北市建築執照記錄


tpebuilding.g0v.ronny.tw

收集台北市從民國 38 年到 102 年的建築使用執照掃描本，目前資料沒有再更新，正在處理其他來源希望可能直接針對全台灣資料做處理

政治獻金切豆腐


campaign-finance.g0v.ronny.tw

利用影像處理將政治獻金的掃描圖檔抓出框線並切出每一個格子，並接力將每一格丟給 @ctiml 做的 http://campaign-finance.g0v.ctiml.tw/ 利用群眾外包來將掃描文件 OCR 出來

GitHub GeoJSON 地圖產生器


github.ronny.tw

將 CSV 或是 GeoJSON 丟到 Github 之後，只需要把網址的 github.com 改成 github.ronny.tw 就可以直接瀏覽結果，因為 Github 是用 js 在瀏覽器畫出來，資料有 1MB 的限制， github.ronny.tw 是在 server 端畫好以 WMS 型式傳回來，因此資料無限制

台北市門牌整理

Fusion Table

從 台北市政府民政局 門牌整合檢索系統 爬下來的台北市門牌資料，一共有 27 萬個門牌和經緯度座標，這個資料是民政局人員人工測量的，因此比 Google Map 的座標更準確，座標是在門牌所在的位置上，例如那種一棟有 40 戶的社區，他的座標也是訂在各自門牌上而不是訂在馬路邊的門口
